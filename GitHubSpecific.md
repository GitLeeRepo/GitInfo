# Source / Text Editing

* **Ctrl-F** - Start searching in file editor
* **Ctrl-G** - Find next
* **Shift Ctrl-G** - Find previous
* **Shift Ctrl-F** - Replace
* **Shift Ctrl-R** - Replace all
* **Alt-G**	- Jump to line
* **Ctrl-Z** - 1Undo
* **Ctrl-Y** - Redo
